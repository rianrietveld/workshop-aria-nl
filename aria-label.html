aria-label voor links

Met een aria-label op links kun je de inhoud overschrijven en er wat duidelijkers van maken.

1: Link, zonder aria-label:
lees meer 

HTML:
<a href="your-url">
	lees meer
</a>

2: Link met aria-label "lees meer over toegankelijke websites":
lees meer 

HTML:
<a href="your-url"
	aria-label="lees meer over toegankelijke websites">
	lees meer
<a>

3. Maar pas op! Een aria-label overschrijft de linktekst:
lees meer 

HTML:
<a href="your-url"
	aria-label="lees meer over bijen houden">
	lees meer
</a>

4. of zonder ARIA maar met CSS .screen-reader-text class
lees meer over bijen houden 

HTML:
<a href="your-url">
	lees meer
	<span class="screen-reader-text"> over bijen houden</span>
</a>

CSS:
	.screen-reader-text {
		border: 0;
		clip: rect(1px, 1px, 1px, 1px);
		clip-path: inset(50%);
		height: 1px;
		margin: -1px;
		overflow: hidden;
		padding: 0;
		width: 1px;
		word-wrap: normal;
	}

Lege links zonder aria-label

Code zonder aria-label:
<a href="link-to-twitter">
	<span class="fab fa-twitter"></span>
</a>

<a href="link-to-instagram">
	<span class="fab fa-instagram"></span>
</a>

Lege links met aria-label

Code met aria-label:
<a href="link-to-twitter" aria-label="Volg ons op Twitter">
	<span class="fab fa-twitter"></span>
</a>

<a href="link-to-instagram" aria-label="Volg ons op Instagram">
	<span class="fab fa-instagram" ></span>
</a>

Nog beter: Zichtbare tekst bij de icoontjes!

Volg ons op:

  • Twitter
  • Instagram
HTML zonder lege links:
<h2>Volg ons op</h2>
<a href="link-to-twitter">
	<span class="fab fa-twitter"></span>
	Twitter
</a>

<a href="link-to-instagram">
	<span class="fab fa-instagram"></span>
	Instagram
</a>
